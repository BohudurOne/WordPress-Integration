# WordPress Integration

Easily integrate **Bohudur Payment Gateway** with your WordPress website using our official plugins.

---

## Available Plugins

### 1. Bohudur Gateway for Tutor LMS  
Plugin: https://wordpress.org/plugins/bohudur-tutor-gateway/  
Documentation: https://docs.bohudur.one/tutorlms/

Integrate Bohudur payment gateway with **Tutor LMS** and accept secure online payments for your courses.

---

### 2. Bohudur Gateway for Easy Digital Downloads (EDD)  
Plugin: https://wordpress.org/plugins/bohudur-gateway-for-edd/  
Documentation: https://docs.bohudur.one/easydigitaldownloads/

Accept payments seamlessly in **Easy Digital Downloads (EDD)** using Bohudur.

---

### 3. Bohudur Gateway for WooCommerce  
Plugin: https://wordpress.org/plugins/bohudur-gateway/  
Documentation: https://docs.bohudur.one/woocommerce/

Enable Bohudur as a payment method in **WooCommerce** stores.
